BookMaps

BookMaps is a peer-to-peer library system that allows neighbors to connect and share their books with each other. Users can list their books on the app and organize borrowing with our integrated chat system. This app was built using:

  • React
  • Redux
  • Node/Express
  • Google Maps API
  • Twilio in-app messaging

Guidr

Guidr is a platform for outdoor guides to compile a list of their adventures. Users can log in, view their trip history, add new trips, and update their personal information. This app was built using:

  • React
  • Redux
  • SCSS
  • JWT Authentication
