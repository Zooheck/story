BookMaps (Full-stack Developer)

BookMaps is a peer-to-peer library system that allows neighbors to connect and share their books with each other. Users can list their books on the app and organize borrowing with our integrated chat system. This app was built using:

  • React
  • Redux
  • Node/Express
  • Google Maps API
  • Twilio in-app messaging
  • GitHub Repo Live Website

JetLogr (Front-end Developer)

JetLoggr helps airline pilots maintain a detailed log of their flight history. Pilots can log in, view dynamic charts depicting their flight history, add new flights, and print their log book with their personal signature. This app was built using:

  • React
  • auth0
  • Material-UI
  • React-Signature-Canvas
  • Github Repo Live Website
